# 自学习多端开发

### 实现功能：

1. 用户：登陆、登出、注册
2. 备忘录：备忘录列表、查看/修改、删除
3. chat：查看历史会话、继续会话、新建会话、删除会话

本项目采取前后端分离、前端采用 uniapp 开发，兼容多端（app 端、小程序、h5）

后端采用 nodejs，数据库借助 mongodb、（后端均在 sealos 上开发，通过 ssh 连接）
